Quick Start (Docker)

git clone https://github.com/cjp-decentralized/cjp-decentralized
cd cjp-decentralized
docker compose up -d

That's it. Your node will:

  • Run a local IPFS node and pin the latest site CID
  • Automatically update when a new CID is published
  • Serve the site on port 8081 as a clearweb fallback
  • Announce itself to the Nostr mirror registry every 15 minutes
  • Expose a Tor hidden service (.onion address printed on first start)

Manual Setup (no Docker)

1. Install IPFS

# Download kubo (go-ipfs) from https://dist.ipfs.tech
ipfs init
ipfs daemon &

2. Pin the site

# Get the latest CID from latest.json in the repo
ipfs pin add <CID>

3. Run the mirror daemon

# Download the binary for your platform from the GitHub releases page
./mirror --ipfs http://localhost:5001

How mirror updates work

The party publishes a signed latest.json file whenever the site is updated. The mirror daemon fetches it every 15 minutes, verifies the Ed25519 signature against a list of trusted party public keys embedded in the binary, and pins the new CID if the signature is valid. A tampered or unsigned update is silently rejected.
